# 🎬 视频转文字 + AI 智能总结

> 把视频扔进去 → 自动出文字 + AI 帮你总结要点。全程一条命令。

## 这个项目是干嘛的？

你有没有过这种场景：看到一个很长的视频（网课、会议录屏、新闻），不想从头看到尾，只想快速知道"讲了什么"？

这个工具帮你做两件事：

1. **听写** —— 把视频里的人声，自动转成文字
2. **划重点** —— 用 AI 大模型读一遍文字，帮你总结出核心要点

```
你的视频.mp4  ──→  完整文字.txt  +  AI总结.txt
```

## 大白话讲原理

### 第一步：视频 → 音频（FFmpeg）

视频文件 = 画面 + 声音，打包在一个文件里。FFmpeg 就像一把剪刀，把声音那根"音轨"单独剪出来，存成 `.wav` 文件。

### 第二步：音频 → 文字（Whisper）

Whisper 是 OpenAI 开源的语音识别模型。它的工作方式很像人：

> 听到声音 → 脑补出可能的字 → 根据上下文选出最合理的那一句

比如它听到 `ji dan ci`，根据上下文判断：你在讲英语学习方法，那应该是"记单词"，不是"鸡蛋词"。

### 第三步：文字 → 总结（Agent）

加了一个 AI Agent，它不只做一次性总结，而是**做完会自己检查、不合格就重写**：

```
🔍 观察 → 读完文字（428 字，讲的是英语学习方法）
🤔 判断 → "这是【讲课】，不是闲聊，也不是新闻"
📋 决策 → "讲课类应该用'知识点提取'格式来总结"
✍️ 执行 → 按策略生成总结
🔬 验证 → 质检员检查——格式对吗？漏了关键信息吗？
   ↓ 不合格？
🔄 重试 → 把质检意见带回提示词，重新写（最多 2 次）
   → 还不过？返回结果但标记"未通过质检"，不糊弄你
```

和普通 AI 调用的区别：不是"问一句、答一句"就完了，而是 **Agent 自己检查自己的输出，不好的话自己改**——这叫"反馈闭环"。

如果视频是新闻，Agent 会用"5W1H"格式总结；如果视频是会议，会用"议题 → 讨论 → 决定 → 待办"的格式。每种类型有对应的质检标准，确保总结格式达标。

## 怎么安装和运行？

### 1. 装 FFmpeg

去 [ffmpeg.org](https://ffmpeg.org) 下载，解压后把 `bin` 文件夹加到系统 PATH。

或者用 winget（Windows 11 自带）：
```bash
winget install Gyan.FFmpeg
```

### 2. 克隆项目

```bash
git clone https://github.com/qilin-19/p2t
cd p2t
```

### 3. 创建虚拟环境 + 装依赖

```bash
python -m venv venv
venv\Scripts\pip install openai-whisper openai

# ★ 如果你有 NVIDIA 显卡，再装 GPU 加速版 PyTorch（快 3~10 倍）
venv\Scripts\pip uninstall torch -y
venv\Scripts\pip install torch --index-url https://download.pytorch.org/whl/cu121
```

### 4. 设置 API Key（AI 总结用）

去 [DeepSeek 开放平台](https://platform.deepseek.com) 注册，创建 API Key，然后在终端里设置：

**如果你用的是 CMD（命令提示符，黑窗口）：**
```bash
set API_KEY=sk-你的密钥
```

**如果你用的是 PowerShell（蓝窗口，提示符是 PS）：**
```powershell
$env:API_KEY = "sk-你的密钥"
```

> 💡 不设置也能跑——只是跳过 AI 总结，转文字功能正常。

### 5. 把视频放进 `videos/` 文件夹，开跑！

```bash
# 批量处理 videos/ 里所有视频
venv\Scripts\python transcribe.py

# 或者只处理指定视频
venv\Scripts\python transcribe.py 记单词.mp4
```

## ✨ 项目亮点

**Agent 自我验证 + 重试闭环。** 普通 AI 调用是"问一句答一句"，答完就完了。这个项目的 Agent 做完总结后会自己调质检员检查——格式对不对、关键信息漏没漏。不合格就带着改进意见重写，最多重试 2 次。实在过不了也会在结果里标注"未通过质检"，不糊弄人。
